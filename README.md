# 百日炼兵计划 - 客户宣传页

弘历软件官方服务频道客户宣传落地页

## 部署说明

本项目使用 GitHub Pages 托管，访问地址：
`https://[你的GitHub用户名].github.io/[仓库名]`

## 首次部署步骤

1. 在 GitHub 创建新仓库
2. Clone 到本地
3. 将本目录内容推送到仓库
4. 进入 Settings → Pages → Source 选择 `main` 分支和 `/ (root)`
5. 等待 1-2 分钟即可访问

## 本地预览

直接用浏览器打开 `index.html` 文件即可

## 后续维护

如需修改页面内容，请联系开发者协助更新。
